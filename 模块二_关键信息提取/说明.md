## 模块二：关键信息提取

**目标**

利用解析后的简历文本，抽取以下关键信息：

- 基本信息（必选）：姓名、电话、邮箱、地址。
- 求职信息（加分）：求职意向、期望薪资。
- 背景信息（加分）：工作年限、学历背景、项目经历。

**主要代码位置**

- `app.py`：
  - `BasicInfo` / `JobInfo` / `BackgroundInfo` 三个 `dataclass`。
  - `extract_basic_info(text: str) -> BasicInfo`
  - `extract_job_info(text: str) -> JobInfo`
  - `extract_background_info(text: str) -> BackgroundInfo`
  - `/api/analyze` 中将上述函数结果组合为结构化 JSON 字段：
    - `basic_info`
    - `job_info`
    - `background_info`

**提取方法简述**

- 使用正则表达式识别手机号、邮箱、姓名等模式。
- 通过包含 “地址 / 现居地 / 居住地” 等关键词的行推测地址。
- 搜索“求职意向 / 期望职位 / 期望薪资”等关键词所在行获取求职与薪资信息。
- 通过“工作经验 / 工作年限”与“本科 / 硕士 / 博士 / 大专”等关键词提取工作年限与学历。
- 通过“项目经验 / 项目经历”分割文本，截取对应段落作为项目经历摘要。
