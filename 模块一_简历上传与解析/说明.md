## 模块一：简历上传与解析

**对应功能**

- 支持上传 **单个 PDF 简历文件**。
- 解析 PDF 多页内容，抽取并清洗纯文本。

**主要代码位置**

- 后端接口：`app.py` 中的：
  - `extract_text_from_pdf`
  - `/api/analyze` 中对上传文件的处理逻辑。
- 前端上传入口：`static/index.html` 中的：
  - `<input type="file" id="file" name="file" accept="application/pdf">`
  - 提交表单 `#analyze-form` 与 `static/main.js` 中的表单提交逻辑。

**使用说明（简要）**

1. 启动后端：
   ```bash
   pip install -r requirements.txt
   python app.py
   ```
2. 浏览器打开 `http://127.0.0.1:5000/`，选择本地 PDF 简历并提交，即会触发本模块逻辑。
